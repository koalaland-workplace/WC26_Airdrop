# WC26 Admin Frontend

Svelte + TypeScript admin panel client for WC26 NFT FANTASY.

## Run

1. From `Viral Activation`:
   - `npm install`
2. Start admin:
   - `npm run dev:admin`

Set API endpoint with:

```bash
VITE_API_BASE=http://localhost:8787
```

Current scaffold includes:

- Telegram login form payload testing
- TOTP verification step
- Session state
- Dashboard metrics fetch
